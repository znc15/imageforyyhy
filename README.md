# 安装说明
推荐使用宝塔安装,建议不要使用虚拟主机安装（估计你也装不上），PHP版本7.2及以上，将整包下载解压（或者直接clone），设置运行目录为/Public,然后配置伪静态，Apache无需配置，nginx伪静态规则在程序根目录有个nginx.txt，复制规则配置即可，之后给所有目录777权限，然后配置/App/Database.php内的数据库信息，还有用户名密码（后台用）,最后导入数据库即可.
